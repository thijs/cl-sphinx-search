Package: cl-sphinx-search

Function run-queries

Lambda List

run-queries (client)

Arguments

  • client -- a sphinx-client

Return Value

nil or a list of hashes

Details

Run the queries added with add-query through searchd.

    (add-query client "test")
    (add-query client "word")
    (run-queries client)


Query searchd with the collected queries added with add-query.

It returns a list of hashes containing the result of each query. Each hash has the following keys:
attributes
: a hash-table containing attributes
fields
: a list of fields
matches
: a hash-table containing the matches
status
: the status returned by searchd
status-message
: the status message returned by searchd
time
: the time searchd took for the query
total
: the total matches returned
total-found
: the total number of matches found
words
: a hash-table containing the matching words with their statistics
