Function set-server

Lambda List

set-server (client &key host port path (host localhost) (port 3312))

Arguments

  • client -- a sphinx-client
  • host -- the host to connect to when using an INET socket
  • port -- the port to connect to when using an INET socket
  • path -- the path to the unix domain socket when not using INET

Return Value

client

Details

Set the server host:port or path to connect to.

    (set-server client :host host :port port)
    (set-server client :path unix-path)


In the first form, sets the host (string) and port (integer) details for the searchd server using a network (INET) socket.

In the second form, where unix-path is a local filesystem path (optionally prefixed by 'unix://'), sets the client to access the searchd server via a local (UNIX domain) socket at the specified path.
