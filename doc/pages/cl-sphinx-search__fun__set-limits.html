Package: cl-sphinx-search

Function set-limits

Lambda List

set-limits (client &key offset limit max cutoff (offset 0) (max 1000))

Arguments

  • client -- a sphinx-client
  • offset -- the offset to start returning matches from
  • limit -- how many matches to return starting from offset
  • max -- maximum number of matches to return
  • cutoff -- the cutoff to stop searching at

Return Value

client

Details

Set the offset, limit, cutoff and max matches to return.

    (set-limits client :limit limit)
    (set-limits client :offset offset :limit limit)
    (set-limits client :offset offset :limit limit :max max-matches)


Set limit of matches to return. Defaults to offset 0 and 1000 max matches.
