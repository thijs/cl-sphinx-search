Package: cl-sphinx-search

Function set-id-range

Lambda List

set-id-range (client min max)

Arguments

  • client -- a sphinx-client
  • min -- minimum id to start searching from
  • max -- maximum id to stop searching at

Return Value

client

Details

Set the id-range to search within (inclusive).

Set the range of id's within which to search. Range is inclusive, so setting [0, 450] both 0 and 450 id's will be found.
